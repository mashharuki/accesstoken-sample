# accesstoken-sample

React + Honoで開発するアクセストークン、リフレッシュトークン学習用のプロジェクト

## 概要

アクセストークン、リフレッシュトークンの仕組みを学習するためのサンプルプロジェクトです。

フロントエンドはReact + Vite、バックエンドは Honoで構築しています。

## 動かし方

### インストール

```bash
pnpm i
```

### フロントエンドの起動

```bash
pnpm frontend dev
```

### バックエンドサーバーの起動

```bash
pnpm backend dev
```
