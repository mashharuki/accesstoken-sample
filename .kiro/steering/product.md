# Product Overview

アクセストークン/リフレッシュトークンの仕組みを学ぶための学習用サンプル。  
フロントエンドとバックエンドを分離し、最小構成で認証関連の流れを把握できることを目的とする。

## Core Capabilities

- フロント/バック分離の学習用構成（React + Hono）
- ローカルでの動作確認に最適化されたシンプルな構造
- 認証トークンの理解に集中できる最小限の機能セット

## Target Use Cases

- 認証トークンの基礎学習や挙動確認
- フロント/バック連携の理解（モノレポでの開発体験）
- 小規模なサンプルをベースにした拡張検証

## Value Proposition

複雑な実装を避け、学習に必要な構成要素だけを抽出しているため、理解と改造が容易。
